# Iceberg Marketing Portal

A premium SaaS Marketing Admin Dashboard built with **React + Vite + Tailwind CSS**.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- **Dashboard** — KPI stats, recent clients, recent campaigns, quick actions
- **Clients** — Add, edit, delete clients with sortable table
- **Email Templates** — Rich text editor with live preview, template variables (`{{name}}`, `{{email}}`, `{{company}}`)
- **Send Campaign** — 3-step wizard: select template → select clients → review & send
- **Campaign History** — Track all campaigns with open rate metrics
- **Settings** — Profile, email config, notifications, security

## Tech Stack

- React 18 + Vite
- Tailwind CSS 3
- Lucide React (icons)
- Inter font (Google Fonts)
